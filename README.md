# Data Science & Machine Learning

A collection of Jupyter notebooks covering Exploratory Data Analysis (EDA) and Feature Engineering techniques.


## Topics Covered

### Exploratory Data Analysis (EDA)
- **Flight Price Prediction** - Analysis of flight pricing data
- **Google Play Store Dataset** - Exploration of app store data
- **Red Wine Dataset** - EDA on wine quality dataset

### Feature Engineering
- **Encoding** - Categorical variable encoding techniques
- **Handling Imbalanced Datasets** - Techniques like SMOTE for class imbalance
- **Handling Missing Values** - Imputation strategies
- **Number Summary and Box Plots** - Statistical summaries and visualization

## Installation

```bash
pip install -r requirements.txt
```

## Requirements

- pandas
- scikit-learn
- imbalanced-learn

## Usage

Open any notebook with Jupyter:

```bash
jupyter notebook
```
